# webgl-experiments

Some WebGL experiments.
___

## Run
- Install `npm install`
- Run `npm start`
- Build `npm run build`
___

## Libraries
- [Three.js](https://github.com/mrdoob/three.js/)
- [ControlKit](https://github.com/brunoimbrizi/controlkit.js) - GUI
- [stats.js](https://github.com/mrdoob/stats.js/) - performance monitor
